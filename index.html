Halaman Web HTML Lanjutan

Header digunakan untuk menampilkan judul atau identitas utama halaman. Bagian ini membantu pengguna mengenali konteks halaman sejak pertama kali membuka. Header juga dapat membantu pengguna memahami tema halaman secara cepat dan meningkatkan navigasi awal.

Navigasi (Navbar)

Beranda Audio Video Iframe Marquee Tabel
Navbar berfungsi sebagai menu navigasi untuk memudahkan pengunjung berpindah antarbagian dalam halaman. Elemen ini meningkatkan pengalaman pengguna agar lebih mudah menemukan informasi. Navbar horizontal menyediakan tautan navigasi ke berbagai bagian halaman, seperti Beranda atau Kontak. Ini meningkatkan kemudahan penggunaan dengan memungkinkan lompatan cepat antar konten. Dalam konteks web, navbar yang responsif dan intuitif sangat penting untuk pengalaman pengguna yang baik, terutama pada situs dengan banyak informasi.

Audio Player

Elemen audio memungkinkan halaman memutar suara atau musik langsung dari browser. Elemen audio memungkinkan penyisipan file suara langsung ke halaman web dengan kontrol seperti play, pause, dan volume. digunakan untuk menambahkan konten multimedia agar halaman web lebih interaktif dan menarik

Video Player

Elemen video digunakan untuk menampilkan konten visual seperti tutorial atau informasi multimedia. Fitur kontrol memudahkan pengguna mengatur pemutaran video, dan memungkinkan untuk penyisipan video langsung di halaman web dengan kontrol pemutaran yang proporsional.

Iframe

Iframe memungkinkan konten eksternal seperti video YouTube atau peta yang dimuat langsung di halaman. Ini memungkinkan integrasi konten eksternal tanpa mengalihkan pengguna dari situs utama. Dalam pengembangan web, iframe berguna untuk memperkaya halaman dengan media interaktif, meningkatkan fungsionalitas tanpa kompleksitas tambahan.

Teks Bergerak (Marquee)

✨ Selamat datang di Pantai Air Manis — Surga Pantai di Kota Padang ✨
Marquee digunakan untuk menampilkan teks berjalan yang menarik perhatian. Marquee membuat teks bergerak secara horizontal atau vertikal, sering digunakan untuk menyoroti informasi penting. Ini menambah elemen dinamis pada halaman. Dalam konteks web, marquee dapat digunakan untuk pengumuman atau iklan, namun harus digunakan bijak agar tidak mengganggu pengalaman pengguna.

Tabel menampilkan data dalam bentuk baris dan kolom agar lebih terstruktur dan mudah dibaca. Tabel digunakan untuk menampilkan data terstruktur dalam baris dan kolom, dengan border-collapse untuk tampilan rapi. Ini membantu menyajikan informasi seperti daftar atau statistik dengan jelas. Dalam pengembangan web, tabel meningkatkan keterbacaan data dan memudahkan analisis, terutama untuk konten informatif seperti laporan atau katalog.
